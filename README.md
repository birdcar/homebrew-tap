# Homebrew Tap for birdcar

This tap contains formulae for tools created by birdcar.

## Installation

```bash
brew tap birdcar/tap
brew install birdhouse
```

## Available Formulae

- **birdhouse** - CLI tool for managing GitHub workflow templates

## Updating

```bash
brew update
brew upgrade birdhouse
```
